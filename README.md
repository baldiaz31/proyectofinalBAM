# Saturno

El proyecto trata de una pagina web centrada en la astrología con relación a la psicología, la espiritualidad. El objetivo es acercar a los usuarios conocimientos y se vera mas adelante de ofrecer cursos de los mismos. 

## Estado del proyecto

En construcción.

## Requisitos

Ninguno. Puede accederse desde una computadora de escritorio, una tablet o un celular. La pagina es responsive.

## Instalación

No requiere instalación, solo conexión a internet.

## Tecnologías utilizadas

Librería Bootstrap y Typed.js
